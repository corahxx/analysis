# 标准化数据产品：七表 UI 设计



本文档描述「标准化数据产品」页面的交互与布局。统计口径仍以 [标准数据产品输出.md](./标准数据产品输出.md) 为准；**各产品表内若另有约定，以本节为准**。



---



## 一、设计说明



| 项目 | 说明 |

|------|------|

| 与旧版差异 | 旧版在加载 `df` 后纵向依次渲染排行榜、运营商、全国、省、城市群、功率、高速、车桩比等，页面冗长。新版仅通过 **7 个按钮**切换，**同一时间只展示一个产品区域**。 |

| **按需加载** | 完成数据来源配置后**不预加载整表**；先展示产品按钮，**点击产品后再加载数据并生成**。库表场景**无「加载该表」步骤**，选好表与表类型即可点产品。 |

| 数据来源 | 「导入文件 / 从数据库选择表」及「请确认表类型」；库表下表类型与表名选择同时可用，无需先加载。 |

| 核心城市群（原 P5） | 未纳入下列七个按钮名称；是否并入「省级数据」或将来单独增加入口，**待业务确认**。当前应用主流程中不再单独展示城市群板块。 |

## 二、UI 线框（文字）



1. **顶部**（保持现有实现）  

   - Banner + 标题「输出标准化产品」  

   - 「数据来源」单选：导入文件 / 从数据库选择表  

   - 上传文件，或库连接 + Schema/表名 + 表类型确认（不预读全表）  



2. **数据源就绪（未点产品前）**  

   - 提示文案示例：`已选择数据来源，请点击下方产品按钮，系统将按需加载数据并生成。`（全表预加载不出现；用户点击某一产品后显示加载中，成功后可在该产品区域下方用辅文展示「已加载 N 行」。）  



3. **产品区**  

   - 小标题：**标准数据产品（按需生成）**  

   - **第一行按钮（7 个）**，文案固定为（顺序可随实现调整，建议与下表映射一致）：  

     `车桩比` · `高速公路` · `功率段分布` · `排行榜` · `全国概况` · `省级数据` · `运营商概况`  

   - 窄屏可采用两行（如 4+3）排列按钮。  



4. **内容区（单块）**  

   - 用户**未点击**任何产品按钮时：`st.info("请点击上方按钮查看对应产品。")`  

   - 点击某一按钮后：`st.spinner` 按需全量读入源表；成功则 `st.caption` 展示「已加载 N 行」；失败则 `st.error` 且不渲染该产品表格。同一数据来源签名未变时，切换产品按钮可复用内存中的 `DataFrame`，不必重复读表。  

   - 仅渲染当前产品对应内容（表格/指标卡/图）及本产品的 **导出 Excel / CSV**（若适用）。

## 四、每张表要求



### 1. 车桩比



**输出**：**一张表**，列名固定为：



| 列名 | 说明 |

|------|------|

| 省份 | 分组维度，使用源表字段 **`省份_中文`**（若无该列则退化为 **`省份`**） |

| 新能源车保有量 | 暂空（占位，后续接数据） |

| 公共充电设施数量 | 按省统计 **数据条数** |

| 车桩比 | 暂空（占位） |

| 评价 | 暂空（占位） |



**交互**：仅展示上述表格 + 导出；生成时机：用户点击「车桩比」后再加载源表并计算。



### 2. 高速公路



**本期**：**占位**。输出列与业务口径**待定**；界面使用 `st.info` 说明「内容待定」，**不提供数据导出**，避免与未定义口径冲突。



### 3. 功率段分布



**分组**：按 **`省份_中文`**（无则 **`省份`**）分省；**无省份列**时整表视为单 Sheet（如 Sheet 名 `全部`）。



**功率列**：充电桩表 **`额定功率`**；充电站表 **`站点总装机功率`**。仅对**可解析为数值**的行分档；无效/缺失功率不计入各档，该省 **总数** = 五档数量之和。



**分档**（单位 kW，与字段数值一致）：



| 功率段 | 条件 |

|--------|------|

| `p < 120kW` | p < 120 |

| `120-250kW` | 120 ≤ p < 250 |

| `250-480kW` | 250 ≤ p < 480 |

| `480-960kW` | 480 ≤ p < 960 |

| `960kW+` | p ≥ 960 |



**每 Sheet 列**：**功率段** | **数量** | **占比** | **环比**  

- **数量**：该省该档 **行数**。  

- **占比**：该档数量 / 该省有效行总数（百分比字符串）。  

- **环比**：空字符串占位。



**交付**：一个 **xlsx**，**每省一个 Sheet**，Sheet 名 = 省份名（需符合 Excel 命名规则并去重）。



**界面**：**省份 `selectbox`** 预览当前省表格；可选柱状图（当前省）；**下载**提供完整多 Sheet 文件。



**计数口径**：桩表、站表均按 **行数** 统计（不按序号/去重站编号）。



### 4. 排行榜



**交付**：一个 **xlsx**，内含 **6 个 Sheet**，名称固定：  

`市场份额榜`、`设施销量榜`、`城市榜Top10`、`星级场站榜`、`型号榜`、`车企私桩榜`。



**公共**：设施类计数按 **数据行数**；运营商用 **`运营商名称`**（无则 **`上报机构`**）；城市用 **`城市_中文`**（无则 **`城市`**）；占位列用 **空字符串**。



| Sheet | 列与规则 |

|-------|----------|

| **市场份额榜** | 运营商、公共充电设施总量（行数）、市场份额（该运营商/全表总量）、环比增速（空）— **全量运营商**，按总量降序 |

| **设施销量榜** | 运营商（去重列出）、新增销量、环比增量、环比增速 — **后三列先空** |

| **城市榜Top10** | 城市、公共充电设施总量、全国占比、环比增速 — Top10 城市，环比空 |

| **星级场站榜** | 运营商（去重）、星级场站数、占比、五星级场站数 — **除运营商外先空** |

| **型号榜** | 设备型号（**充电桩型号**）、装机量（行数）；**仅桩表**有数据，站表仅表头 |

| **车企私桩榜** | 车企名称、私桩安装量、占比、环比增速 — **整表先空**（仅表头） |



**界面**：**Sheet 下拉**预览其中一张表；**下载**为上述六 Sheet 合一文件。



### 5. 全国概况

**交付**：一个 **xlsx**，内含以下 **Sheet**（名称固定，顺序如下）：

`充电站` · `公共充电桩` · `交流桩` · `直流桩` · `充电电量` · `共享私桩` · `换电站` · `换电电量` · `私桩及个人充电设施` · `随车配建` · `交直流桩`

**公共列**（有数据的表均使用）：**省份** | **数量** | **全国占比** | **环比增速**  
- **数量**：按省 **数据行数**（`groupby` 省份后 `size`）。  
- **全国占比**：该省数量 / **当前 Sheet 所基于的子集总行数**（百分比字符串）。  
- **环比增速**：暂 **空字符串**。

**分表规则**

| Sheet | 说明 |
|-------|------|
| **充电站** | 四列先 **占位**：仅表头，无数据行。 |
| **公共充电桩** | **仅充电桩表**：按 **`省份_中文`**（无则 **`省份`**）分组统计行数、全国占比；环比空。**充电站表**无此口径时为仅表头。 |
| **交流桩** | 与公共充电桩相同，且筛选 **`充电桩类型_转换`**（去空格后）**等于 `交流`**。 |
| **直流桩** | 同上，**等于 `直流`**。 |
| **充电电量** | 先空（仅表头）。 |
| **共享私桩** | 先空（仅表头）。 |
| **换电站** | 先空（仅表头）。 |
| **换电电量** | 空（仅表头）。 |
| **私桩及个人充电设施** | 空（仅表头）。 |
| **随车配建** | 空（仅表头）。 |
| **交直流桩** | 与公共充电桩相同，且 **`充电桩类型_转换` = `交直流`**。 |

**界面**：**Sheet 下拉**预览其中一张表；**下载**为上述多 Sheet 合一文件（与排行榜交互一致）。不再单独强调顶部三张指标卡；若保留简要汇总可在实现中酌定。



### 6. 省级数据

**分组**：按 **`省份_中文`**（无则 **`省份`**）；**每省一个 Sheet**，Sheet 名 = **省份名称**（Excel 命名需合法、去重，与功率段多 Sheet 规则一致）。

**每个 Sheet 表结构**（三列）：

| 数据维度 | 数值 | 环比变化 |
|----------|------|----------|

**数据维度** 固定行顺序（共 11 行）：

`充电站` · `公共充电桩` · `交流桩` · `直流桩` · `充电电量` · `共享私桩` · `换电站` · `换电电量` · `私桩/个人充电设施` · `随车配建` · `交直流桩`

**填充规则**（**仅充电桩表** `for_pile=True`；**充电站表**下各指标 **数值、环比变化** 均空，仍输出上述 11 行）：

| 数据维度 | 数值 | 环比变化 |
|----------|------|----------|
| 充电站、充电电量、共享私桩、换电站、换电电量、私桩/个人充电设施、随车配建 | **空** | **空** |
| 公共充电桩 | 该省 **数据行数** | **空** |
| 交流桩 | 该省且 **`充电桩类型_转换`**（strip）**= 交流** 的行数 | **空** |
| 直流桩 | 同上，**= 直流** | **空** |
| 交直流桩 | 同上，**= 交直流** | **空** |

**交付**：一个 **xlsx**，多 Sheet（每省一表）。

**界面**：**省份 `selectbox`** 预览当前省上述三列表；**下载**完整多 Sheet 文件（与功率段、全国概况一致）。



### 7. 运营商概况

**交付**：一个 **xlsx**，内含以下 **Sheet**（名称与顺序固定）：

`公共充电设施` · `共享私桩` · `公用充电桩` · `专用充电桩` · `直流桩` · `交流桩` · `三相交流桩` · `充电功率` · `充电电量` · `充电站` · `换电站`

**有数据 Sheet 的统一列**：**运营商** | **数值** | **环比变化** | **环比增速**  
- **运营商**：分组键为 **`运营商名称`**（无则 **`上报机构`**）。  
- **环比变化**、**环比增速**：本期均 **空字符串**。

**分 Sheet 规则**（**仅充电桩表** `for_pile=True` 且存在运营商列时计算；否则对应 Sheet 为 **仅表头**。**充电站表**下各 Sheet 均为仅表头占位）：

| Sheet | 规则 |
|-------|------|
| **公共充电设施** | 按运营商分组，**数值** = 该运营商 **数据行数**（全表行数口径），按数值降序。 |
| **直流桩** | 先筛选 **`充电桩类型_转换`**（strip）**= `直流`**，再按运营商分组统计 **行数**，降序。 |
| **交流桩** | 同上，**= `交流`**。 |
| **充电功率** | 按运营商分组，对 **`额定功率`** 数值化后 **求和**，**数值** 为功率合计，降序。 |
| **共享私桩、公用充电桩、专用充电桩、三相交流桩、充电电量、充电站、换电站** | **仅表头**（本期空）。 |

**界面**：**Sheet 下拉**预览；**下载**为上述多 Sheet 合一文件（与全国概况、排行榜一致）。

## 五、实现备注（开发者）



- `app.py` 使用 `st.session_state["product_panel"]` 记录当前选中的按钮键（字符串，与上表「按钮名称」一致）。  

- 切换数据来源或重新加载 `df` 时，可考虑将 `product_panel` 置为 `None`，避免旧面板状态误解（可选）。  

- **大表性能（百万～千万行）**  
  - 库表读取 **禁止默认 `SELECT *`**：`db_helper.read_table` 支持 `columns`，产品页通过 `get_table_column_names` + `handlers/product_slim_columns.py` 与表结构求交集后只拉 **九列以内**（桩/站各一套候选列），显著降低 IO 与 pandas 解析时间。  
  - 可选环境变量 **`ANALYSIS_READ_CHUNKSIZE`**（如 `80000`）：分块 `read_sql` 再 `concat`，降低单次内存峰值（总耗时可能略增）。  
  - **一键生成七表**：按钮「一键生成七表（ZIP）」对应 `product_panel == "__ALL_SEVEN__"`，在 **同一份已缓存 `df`** 上调用 `handlers/product_bundle.py` 打包 7 个 xlsx，**不重复查库**。  
  - 数据库侧可对 **`省份_中文`、`运营商名称`、`充电桩类型_转换`** 等高频分组列建索引，进一步缩短聚合时间（应用层仍为 pandas 聚合，后续可再改为 SQL 聚合下推）。
